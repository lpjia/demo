各元素有啥特性, 检查元素

块元素, 宽度默认100%, 占满一行, 高度默认是内容高度, border-box下是边框+padding+ctt, padding margin 上下左右都生效
行内块, 宽度默认内容宽度, 横向排列, 高度默认是内容高度, border-box下是边框+padding+ctt, padding margin 上下左右都生效
行内块, 宽度默认内容宽度, 横向排列, 高度默认是内容高度, border-box下是边框+padding+ctt, padding margin 上下左右都生效
块元素, 宽度默认100%, 占满一行, 高度默认是内容高度, border-box下是边框+padding+ctt, padding margin 上下左右都生效
行内块, 宽度默认内容宽度, 横向排列, 高度默认是内容高度, border-box下是边框+padding+ctt, padding margin 上下左右都生效
span行内元素, 宽度默认内容宽度, 横向排列, 除 margin 上下不生效, 宽高不可以指定 span行内元素, 宽度默认内容宽度, 横向排列, 除 margin 上下不生效, 宽高不可以指定 padding 上下生效, 却占用
